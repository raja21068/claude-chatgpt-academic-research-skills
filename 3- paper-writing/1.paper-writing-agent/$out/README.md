# Output Directory Structure

All generated outputs from the paper-writing skill go here.

```
$out/
├── drafts/    ← Section drafts and full paper drafts
├── tables/    ← Generated result tables (LaTeX/Markdown)
├── figures/   ← Generated figure code and exported images
└── reports/   ← Audit reports, reviewer simulations, checklists
```

Organize your outputs into these subdirectories to keep things clean.
